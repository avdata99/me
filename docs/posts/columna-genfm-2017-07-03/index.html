OpenDataCordoba tuvo durante 2017 una columna semanal en el programa Tarde para Chequear de Radio GenFM 107.5 sobre datos abiertos y gobierno abierto. El 3 de julio de 2017 me tocó a mí ser el invitado de la columna.

Tema: "Carta marina a todas las escuelas que se van a usar para todas las elecciones".

Radios municipales en Córdoba (2014)

Andres Vazquez

2014-05-15 12:00

Comentarios

Código fuente

Trabajo realizado en mayo de 2014 sobre los radios municipales definidos y en conflicto en la provincia de Córdoba, montado como capa de un mapa interactivo a nivel de radio censal del Censo 2010.

El radio municipal es el límite legal del territorio que cada municipio cordobés puede administrar. La definición y delimitación de varios de estos radios estaba (y sigue, en muchos casos) sin resolver, generando conflictos entre municipios vecinos y zonas grises de jurisdicción.

La aplicación que mostraba esa capa era parte del proyecto Córdoba según el Censo 2010, surgido del Hackathon FOPEA 2014. Su código quedó abierto en GitHub — HackathonFOPEA/CensoCordoba2010.

Nota: la aplicación interactiva (cordoba-argentina-segun-censo-2010/) ya no está disponible en este sitio. El código fuente sigue en GitHub para quien quiera revivirla.

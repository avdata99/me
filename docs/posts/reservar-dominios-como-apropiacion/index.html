Reservar dominios como forma de apropiación

Andres Vazquez

Comentarios

Análisis presentado durante el NICathon (marzo 2015).

Reserva de dominios en NIC.ar

Como parte del #NICathon se listan a continuación dominios web argentinos que han sido reservados (son intentos de compra con formas de pago no automáticas) multiples veces.

Las reservas son algo necesario para aquellos que no pueden hacer pagos por internet. La necesidad de brindar este servicio ha sido aprovechada por personas que consecutivamente amagan la compra para seguir en control de un dominio hasta conseguir un comprador.

NIC Argentina atiende los reclamos en este sentido cuando se hacen forzando a los titulares de las reservas (que lamentablemente no se informan) a confirmar o liberar el dominio. Si bien estos reclamos son escuchados no parece existir una actitud preventiva que se efectiva hasta el momento.

La lista a continuación es segun el estado de los dominios al 3 de marzo de 2015. Si desean conocer mas datos de cada dominio pueden remitirse a la aplicación web con datos actualzados: ArNicApp.

Esta lista muestra que esta práctica se realiza con frecuencia y es entonces un método usado para acaparar dominios en Argentina sin pagar por ellos. Incluso existe una herramienta que revisa todos los días los dominios re-reservados.

Visualización interactiva (DataTable con búsqueda y filtros): /data/Reservar-dominios-como-forma-de-apropiacion/

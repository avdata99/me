<p>Jorge es diseñador y desarrollador de software en Zaragoza, España. Es además lo que llamaríamos en Argentina un activista o hacker cívico que tiene numerosos proyectos propios.</p>

<h3>Ideas del episodio</h3>
<ul>
  <li>Jorge comenzo a trabajar con datos desde el primer portal de Aragón.</li>
  <li>Bislai es un proyecto de análisis de datos sobre el debate legislativo. Los datos publicados eran muy pobres y con este proyecto los datos se hicieron más accesibles y con la posibilidad de buscar.</li>
  <li>Bislai crea un dataset con el resumen de todas las votaciones que permite generar estadísticas.</li>
  <li>La base de datos de los códigos postales esta cerrado desde 2017 (pagando se puede acceder)</li>
  <li>Con los datos históricos de clima puede verse como afecta el cambio climático a tu ciudad.</li>
  <li>Ahora Jorge esta ampliando este análisis del clima de Zaragoza para toda España.</li>
  <li>Jorge estaría dispuesto a ser funcionario de Gobierno.</li>
  <li>El gobierno a veces se saca la foto luego de liberar el portal y después no le dan todo el seguieminto que se requiere.</li>
  <li>Hay formulario (bastante feo según Jorge) online de pedidos de acceso a la información pública.</li>
  <li>Cuando los activistas no conocen a ley a veces caen en respuestas del gobierno de que no deben publicar algo cuan si deben.</li>
  <li>Hay periodismo de datos en España pero no particularmente en Zaragoza y Aragón.</li>
  <li>Jorge pide más y mejor comunicación del gobierno hacia os usuarios de los datos.</li>

</ul>

Esto es interesante porque muestra la importancia de usar estándares y software libre.

QGIS es un software libre para analizar y procesar datos espaciales (de esos mapitas lindos que vemos). Además permite crear extensiones que le agreguen funcionalidad.

Eso (al parecer) es lo que hicieron en Kagis, Austria, y después continuó Toronto, Canadá. Colaboraron en la construcción de una extensión de QGIS que permite mostrar recursos alojados en portales de datos abiertos hechos con CKAN.

![Extensión de QGIS para CKAN](/img/twitter/software-libre-conecta-gobiernos/kagis-toronto-qgis-a.jpg)

![Extensión de QGIS para CKAN](/img/twitter/software-libre-conecta-gobiernos/kagis-toronto-qgis-b.jpg)

CKAN es software libre e incluye un servicio de datos (una API) que permite a otras aplicaciones reutilizar su contenido.

Kagis y Toronto eligieron pagar por desarrollo que necesitaban y ahora la herramienta puede recibir mejoras desde los usuarios de todo el mundo que la usan y la mejoran.
